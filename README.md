## 📁 Dataset

| Champ | Description |

| Country | Pays |

| Product_Category | Catégorie produit |
| Product | Produit |
| Sales_Channel | Online / Store |
| Units_Sold | Quantité |
| Unit_Price | Prix |
| Revenue | Chiffre d’affaires |
| Customer_Type | New / Returning |
